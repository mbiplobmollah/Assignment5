Question-1: What are the different ways to select an element in the DOM?

There are quite a few ways to select and element in the DOM. Some of the ways are :{1. getElementById()}, 2. getElementsByClassName(), 3. getElementByTagName(), 4. querySelector(), 5. querySelectorAll, 6. getAttribute().

Question-2: What is the difference between innerHTML, innerText, and textContent ?

1. In innerHTML you can easy easily add many HTML tag you can also add inline design in this tags which makes it amazing. 2. In innerText you can only add or see the visible content of an element. It will not display HTML tags. 3. In textContent is is similar to innerText you can add or see text content of an element but it will display hidden contents as well.

Question-3: What is event delegation in the DOM?

Event delegation is a technique in which we add a single event listener to a parent element rather thant adding event listeners to indivisual child elements.

Question-4: What is event bubbling in the DOM?

Event bubbling is a type of event propagation in the DOM where and event starts from the target element and then bubbles up to parent elements in the DOM tree. It will trigger any relevant event listeners along the way.

Question-5: How do you create, add, and remove elements using JavaScript?

1. Create an element: const newDiv = document.createElement('div'); 2. Add an element: const parentElement = document.getElementById('parent'); parentElement.appendChild(newDiv); 3. Remove an element: newDiv.remove(); This is how we create, add and remove elements in JavaScript.
